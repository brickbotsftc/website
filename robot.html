Robot Components

Breaking down the key systems and mechanisms

Drivetrain

Mecanum wheel drive system providing omnidirectional movement for maximum maneuverability on the field.

  • Type: Mecanum Drive
  • Motors: 4x REV HD Hex
  • Top Speed: 1.2 m/s

Intake System

Custom-designed intake mechanism for reliable game element collection with adjustable speed control.

  • Type: Roller Intake
  • Motor: 1x REV Core Hex
  • Width: 14 inches

Lift Mechanism

Vertical lift system with precise positioning for accurate scoring at multiple heights.

  • Type: Linear Slides
  • Motors: 2x REV HD Hex
  • Extension: 36 inches

Scoring Mechanism

Precision scoring system designed for quick and accurate placement of game elements.

  • Type: Claw Gripper
  • Servo: 1x High Torque
  • Grip Force: 8 lbs
